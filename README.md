## Commands
| Command    | Result                                                     |
| ---------- | ---------------------------------------------------------- |
| `!`        | Print "Hello, World!" in the console                       |
| `@`        | Init `counter` with default value (0) / set `counter` is 0 |
| `>`        | Increment counter by 1                                     |
| `<`        | Decrement counter by 1                                     |
| `?`        | Print value of `counter` to console                        |
